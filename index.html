Step 1 of 4

Create Your HTML File

What you need
A single index.html file
Your website lives in one file called index.html. This is the entry point — Vercel (and every web server) looks for this file automatically when someone visits your URL.

It can contain anything: a CV, a portfolio, a landing page, a presentation.

Step 2 of 4

Push to GitHub

Version Control
Commit your file and push to a GitHub repository
Git tracks every change you make. GitHub stores your code in the cloud so Vercel can access it. Every time you push, Vercel sees the update and redeploys automatically.
# Stage your file
git add index.html

# Save a snapshot
git commit -m "Update website"

# Send to GitHub
git push origin main

Step 3 of 4

Connect Vercel to GitHub

One-time setup
Import your GitHub repo into Vercel
Do this once and you're done. Vercel watches your repo and deploys every new push automatically.
1️⃣

Log in or sign up free
2️⃣

Click the button on the dashboard
3️⃣
Import your repo
Select from GitHub list
4️⃣
Click Deploy
No config needed for static sites
